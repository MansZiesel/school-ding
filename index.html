Deze website heb ik gemaakt omdat ik van school de opdracht had gekregen om een weksite te maken. De opdracht was: Maak eerst een opzet van de code voor je pagina een website die ten minste uit drie pagina's bestaat. De pagina’s horen op elkaar te lijken, ze mogen alleen niet dezelfde opzet hebben. Je mag maximaal één stylesheet gebruiken en deze moet extern zijn. ook moet je minimaal drie plaatjes verwerken in je website
