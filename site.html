Ik heb het volgende gedaan om deze website te maken: Om te beginnen heb ik een ontwerk bedacht, dit kunt u rechts zien. Daarna moest ik van de opdracht alle Block level elements in een word document zetten. Ook moesten in dit document alle classes en id's gezet worden. Dit document kunt u Hier downloaden. Hierna ben met de site begonnen, ik heb de navbar eerst gemaakt, daarna had ikop internet een mooie achtergrond gezocht deze heb ik aangepast en geimplementeerd. Ook heb ik de kolommen gemaakt. en tot slot hier een daar nog wat mooie effecten zoals shadows en select kleuren, daarna was ik klaar.
